# COSMO FINDER - Web Frontend

Modern ve şık bir fal okuma platformunun React + Vite ile geliştirilmiş web arayüzü.

## 🚀 Özellikler

- ✨ **Responsive Tasarım** - Tüm cihazlarda mükemmel görünüm
- 🎴 **7 Fal Türü** - Tarot, Yıldızname, Kahve Falı, Rune, Çin Falı, Günlük Burç, Kabala
- 💳 **Stripe Ödeme Entegrasyonu** - Güvenli premium abonelik
- 🔐 **JWT Kimlik Doğrulama** - Güvenli kullanıcı yönetimi
- 🎨 **Tailwind CSS** - Modern ve özelleştirilebilir tasarım
- 📱 **Mobile First** - Mobil cihazlar için optimize edilmiş
- 🌙 **Dark Mode** - Göz yormayan koyu tema

## 🛠️ Teknoloji Stack

- **Framework:** React 18 + Vite
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Routing:** React Router
- **Icons:** Lucide React
- **Payment:** Stripe

## 📦 Kurulum

```bash
npm install
npm run dev
```

## 🌐 Bağlantılar

- Backend API: http://localhost:5050
- Frontend: http://localhost:5173
